# Glossary

[← References](references.md) · [Home](index.md)

---

| Term                  | Definition                                                                                                                |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| **Platform**          | Orchestration layer that connects clients and services                                                                    |
| **Intent**            | A request from one app for another app to perform an action on a given data type                                          |
| **Capability**        | An app's declaration of its ability to handle a given type of intent                                                      |
| **Intent filter**     | Synonym for capability in the Android vocabulary                                                                          |
| **Client**            | The app that initiates the intent                                                                                         |
| **Service / Target**  | The app that resolves the intent (provides the service)                                                                   |
| **Registry**          | Storage for the manifests — and therefore the capabilities — declared by services                                         |
| **Resolver**          | Mechanism that matches an intent with the available capabilities. The user may need to choose when multiple matches exist |
| **Shell app**         | A "shell" app that wraps a service inside a platform app                                                                  |
