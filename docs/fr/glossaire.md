# Glossaire

[← Bibliographie](bibliographie.md) · [Accueil](index.md)

---

| Terme                | Définition                                                                                                                         |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Plateforme**       | Couche d'orchestration qui met en relation clients et services                                                                     |
| **Intent**           | Requête d'une app pour qu'une autre app réalise une action sur un type de données                                                  |
| **Capability**       | Déclaration par une app de sa capacité à gérer un type d'intent                                                                    |
| **Intent-filter**    | Synonyme de capability dans le vocabulaire Android                                                                                 |
| **Client**           | L'app qui initie l'intent                                                                                                          |
| **Service / Cible**  | L'app qui résout l'intent (fournit le service)                                                                                     |
| **Registre**         | Stockage des manifestes et donc capabilities déclarées par les services                                                            |
| **Résolveur**        | Mécanisme qui matche un intent avec les capabilities disponibles. L'utilisateur peut avoir à choisir dans le cas de match multiple |
| **Coquille**         | App « shell » qui encapsule un services dans une app plateforme                                                                    |
